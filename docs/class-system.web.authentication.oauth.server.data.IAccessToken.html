Interface IAccessToken

Represents an access token entry, that may be stored in a database and should atleast be associated with a resource owner, a client, and have an expiration date.

From RFC 6749 - The OAuth 2.0 Authorization Framework:

ACCESS TOKEN Access tokens are credentials used to access protected resources. An access token is a string representing an authorization issued to the client. Tokens represent specific scopes and durations of access, granted by the resource owner, and enforced by the resource server and authorization server.
