Class OAuthTokenGrantException

The OAuthTokenGrantException is intended to be thrown within the token endpoint logic of the OAuth Provider or the Authorization Server.

By making use of this exception class, the provider will be able to properly handle any errors and report back to the client in a matter that conforms with the standards defined by the OAuth 2.0 specification.
