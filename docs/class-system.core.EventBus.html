Class EventBus

A centralized event bus capable of having handlers registered to events raised by multiple class instances that might not even exist and the moment.

Namespace: system\core
Author: Lumina Framework lumina@incubator.neticle.com
Since: 0.2.0

Registers a new event handler for the specified event, regardless of which object it was raised from.

Registers a new event handler for the specified event, regardless of which object it was raised from.

Parameters

$event
The name of the event to register the handler for.
$callback
A callable reference to the function handling this event. <p>The function signature depends on the event it is supposed to handle and you should look at the element documentation to find out exactly what it is.</p> <p>The callback function should always return TRUE, unless it needs to stop propagation or cancel the event, in which case FALSE should be returned.</p> <p>Do not forget NULL evaluates to FALSE, meaning that if no return statement is specified no more handlers will be called and in some cases the event will be canceled.</p>
