The raw cookie value.
$expiry
<p>The number of seconds the cookie will expiry in since the moment it was defined, or 0 (zero) for a session cookie.</p>
$path
<p>The cookie path. By default, the current request path will be used if no value is specified.</p>

<p>The cookie domain. By default, the current domain will be used if no value is specified.</p>

<p>A flag indicating wether or not the cookie should only be included in secure requests, through HTTPS/SSL.</p>
$http
<p>A flag indicating wether or not the cookie should only be available in the request headers. This means when it's set to TRUE (default), the browser will not grant access to this cookie to any underlying scripts (e.g.: javascript).</p>
