# getBaseModel( string $class = null, string $context = 'default' )

Returns base instance of a model of the specified class after resetting it to the given context.

Returns base instance of a model of the specified class after resetting it to the given context.

This function will re-use a model instance that was previously created for this purpose, which means you can not use a nested instance of the same class.

If nested instances of the same model are required you can simply create them yourself (new MyModel($context, ...)).

This is simply an utility method that needs to be reflected by each model class, allowing you to use your models without having to manually create a new instance each time.

The name of the class to get the model of.
