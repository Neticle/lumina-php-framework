Class PasswordDigest

The message digest component will apply a random salt to a given password and hash it according to certain options, trough PHP's 'crypt' implementation.

public
# setSalt( string $salt )

Defines a static preffix salt to be applied to the raw passwords in order to make the generation of rainbow tables exceptionally hard, even when the attacker gains access to the original hash with lower cost.

Defines a static preffix salt to be applied to the raw passwords in order to make the generation of rainbow tables exceptionally hard, even when the attacker gains access to the original hash with lower cost.

Please note that in addition to any defined salt, a random salt is always generated and used when digesting passwords. Unlike that random salt, this static salt will not be contained in the final hash as it's applied to the passwords directly.

Parameters

$salt
The static password salt.
