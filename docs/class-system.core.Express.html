# configure( array $configuration )

Configures the current instance by invoking the matching 'set' methods for each one of the given properties.

Configures the current instance by invoking the matching 'set' methods for each one of the given properties.

If the 'set' method is not found or invalid the property will be ignored and a NOTICE will be raised.

The 'set' method of a property is calculated by concatenating 'set' with the property name (first letter in upper case).
